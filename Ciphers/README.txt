README

Project: Ciphers

Description:
	The program contains two types of ciphers, a Caesar shift cipher 
		and a date scramble cipher.

Files:
	Makefile
	caesarCipher.hpp
	dateCipher.hpp
	simpleString.hpp
	ioutils.hpp
	
	caesarCipher.cpp
	dateCipher.cpp
	simpleString.cpp
	ioutils.cpp
	test-caesar.cpp
	test-date.cpp
	
	input.txt
	
Execution:
	Execute Makefile with command "make" followed by "./test-caesar input.txt"
		or "./test-date input.txt".
	Program will encrypt and decrypt the sample text.
